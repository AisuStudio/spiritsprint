# Spirit Sprint

Self-Service Web-Flow für den **Aisu Spirit Sprint** — ein strategisches
Ideations-Workshop-Framework von Aisu.Studio (Soul → Skills → Strategy →
Spirit), bislang ein Facilitator-Deck (`AS_ASS_2_CIB_Example.pdf`).

Ziel: dasselbe Framework als eigenständiges Tool, das ein Team ohne
Moderator:in durchlaufen kann.

- [`concept.md`](./concept.md) — Schwachstellenanalyse des Original-Decks +
  vollständige Flow-Spezifikation
- [`design/aisukurimu-tokens.css`](./design/aisukurimu-tokens.css) —
  Design-Tokens, übernommen aus
  [AisuStudio/aisukurimu](https://github.com/AisuStudio/aisukurimu)
- [`prototype/index.html`](./prototype/index.html) — klickbarer
  Konzept-Prototyp (kein Build nötig, direkt im Browser öffnen)
